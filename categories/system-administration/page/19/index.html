Vmware: RVTools

This sly bastard wrote an excellent little .NET app for enumerating esx farm information. Any sysadmins working with VMware should definitely check this out. I really like the vHealth tab which will show possible zombie vmdks. Add the ability to export to csv and you have a nice tool for on the fly farm enumeration for the boss. Go get RVTools now!
